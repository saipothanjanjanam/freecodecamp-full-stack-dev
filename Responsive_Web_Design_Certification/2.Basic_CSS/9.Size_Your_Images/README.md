# Size Your Images

### DESCRIPTION

* We can also apply reduce the images width using CSS property`width` like sizing the fonts using CSS property `font-size`. Below code is used to reduce the image to `100px`(pixels) wide.
```html
<style>
    .small-image{
        width: 100px;
    }
</style>
```
