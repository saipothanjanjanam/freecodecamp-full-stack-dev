# Use An ID Attribute To Style An Element

### DESCRIPTION
* `id` in `style` tags is implemented by `#` like `.` for `class`,
```html
    <style>
        #some-id{
            background-color: red;
        }
    </style>
    <h2 id="some-id"></h2>
```
