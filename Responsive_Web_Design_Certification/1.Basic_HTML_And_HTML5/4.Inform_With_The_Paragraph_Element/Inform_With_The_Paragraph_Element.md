# Inform With The Paragraph Element

### DESCRIPTION
* `<p>` is the element used for writing paragraphs in HTML.
* `<p>` and `</p>` are opening and closing tags.

### CODE

```html
<h1> Hello World </h1>

<h2> CatPhotoApp </h2>

<p> Hello Paragraph </p>
```

### OUTPUT

<h1> Hello World </h1>

<h2> CatPhotoApp </h2>

<p> This is paragraph</p>
