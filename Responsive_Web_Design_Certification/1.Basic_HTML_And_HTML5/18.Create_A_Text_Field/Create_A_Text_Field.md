# Create A Text Field

### DESCRIPTION
* User input is one of the way to get information/data from the user.
* We can create text field in HTML using `input` tag.
* Input tag is self closing tag.

### CODE
```html
<h2>CatPhotoApp</h2>
<main>
    <p>Click here to view more <a href="#">cat photos</a>.</p>
    <a href="#"><img src="https://bit.ly/fcc-relaxing-cat" alt="A cute orange cat lying on its back."></a>
    <p>Things cat love:</p>
    <ul>
        <li>cat nip</li>
        <li>laser points</li>
        <li>lasagna</li>
    </ul>
    <p>Top 3 things cats hate:</p>
    <ol>
        <li>flea treatment</li>
        <li>thunder</li>
        <li>other cats</li>
    </ol>
    <input type="text">
</main>
```
